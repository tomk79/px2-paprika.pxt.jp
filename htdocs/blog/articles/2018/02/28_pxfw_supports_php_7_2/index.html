2018年2月28日、 Pickles Framework と プラグイン 1件 がリリースされました。この更新により、 PHP 7.2系 で起きていた不具合が解決されます。

composer update を実行して、更新することをおすすめします。

フレームワーク

pickles2/px-fw-2.x 2.0.37

  • PHP 7.2 で SCSS のプレビューが失敗する問題を修正。
  • 依存ライブラリ michelf/php-markdown, leafo/scssphp のバージョンを更新。

pickles2/px-fw-2.x 2.0.36

  • PHP 7.2 で、Warning が発生する問題を修正。

プラグイン

pickles2/px2-px2dthelper 2.0.7

  • 依存ライブラリ michelf/php-markdown, leafo/scssphp のバージョン制約を緩和。 (pickles2/px-fw-2.x の更新に合わせられるように)
  • $conf->px2dtconfig が連想配列で設定されている場合に、正規化処理が適切に反映されない不具合を修正。
